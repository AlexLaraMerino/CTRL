# CTRL — MVP Scope

## Goal
Deliver a working iPad daily map with persistent state and fast touch interaction.

## In Scope
- Map
- Date selector
- Notes
- Employees
- Work sites
- Copy day
- Copy yesterday
- Apply to week
- Local persistence

## Out of Scope
- ERP features
- permissions
- analytics
- backend sync
- documents and PDF annotation in this first native milestone

## Acceptance
User can:
- switch dates
- see persistent daily state
- move employees on the map
- hide and show work sites
- copy configurations
- use the app fluidly on iPad

## Rule
Protect simplicity and interaction quality.
